# 关于

这是基于 Jekyll 部署的静态 Blog, 采用基于 [jekyll-theme-H2O](https://github.com/kaeyleo/jekyll-theme-H2O)
的 [jekyll-theme-H2O-ac](https://github.com/zhonger/jekyll-theme-H2O-ac) 主题。

主要用于收集网络上的文章、以及自己的学习笔记、排错日志、日常日记等。

如果有侵权之处，请联系删除。
